Fusion Score Weights

  • GNN / ML Score 30 %
  • Behavioral Score 25 %
  • Graph (Propagation) Score 25 %
  • Cluster Score 20 %

Flow Score Weights

  • Flow Value (normalised) 25 %
  • Path Length 20 %
  • Avg Node Risk 20 %
  • Flow Direction 20 %
  • Time Gap Speed 15 %

Pipeline Steps

  • Fetch via Alchemy API
  • Normalize raw transfers
  • Expand to neighbours (non-fast)
  • Build NetworkX DiGraph
  • Compute 15+ graph features
  • Serialize to JSON

Key Thresholds (src/config.py)

  • Risk HIGH > 0.70
  • Risk MEDIUM > 0.40
  • Flow: laundering > 0.70
  • Flow: suspicious > 0.40
  • Propagation α 0.30
  • Propagation rounds 5

src/config.py — Key Parameters

RISK_HIGH_THRESHOLD      = 0.7     # final score → HIGH risk
RISK_MEDIUM_THRESHOLD    = 0.4     # final score → MEDIUM risk

BEHAVIORAL_TX_HIGH       = 50      # tx count → high behavioral score
BEHAVIORAL_TX_MED        = 20
BEHAVIORAL_VALUE_HIGH    = 5       # avg ETH value → high
BEHAVIORAL_VALUE_MED     = 1
